== Description ==\
\
Project 1 has examples of classes that are groundwork to learning the foundations\
of Java. \
\
Money: A class that has functions to add/remove coins to an account. Also has the\
	 function to display the total amount of money in the minimum amount of \
	 individual coins.\
Counter: This class has functions to have a counter that can be incremented up \
	    until the limit is reached where it will automatically return to 0. \
	    Also has the ability to reset on command and unrest on command.\
Rectangle: A class that defines a rectangle, from there you can request the width\
		length, perimeter, and area of said rectangle.\
String: This class displays the ability to display information from a string\
Temperature: This class has the ability to set temperatures in Fahrenheit or \
		  Celsius, then add them correctly and display them in Celsius. \
\
